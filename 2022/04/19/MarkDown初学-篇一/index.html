1. 标题

嗯，这个不难，语法如下：

  • # 一级标题
  • ## 二级标题
  • ### 三级标题
  • ……
  • #### 六级标题

貌似只有六级标题。效果如下：

  • 一级标题

  • 二级标题

  • 三级标题

  • ……

  • 六级标题

敲重点：这里的”#“后有空格，加在最后一个”#”号后面（ … ）

2. 倾斜、加粗、倾斜加粗

语法如下：

  • _倾斜_ or *倾斜*

  • __加粗__or**加粗**

  • ___倾斜加粗___***倾斜加粗***

  • ~~删除~~

    效果如下：

  • 倾斜

  • 加粗

  • 倾斜加粗

  • 删除

敲重点：这里的符号都没有空格( 就很离谱 )

3. 引用

语法如下：

  • > 引用

  • >> 多层嵌套( 貌似是这个名词 )引用

  • >>>你还可以加很多层( …够了 )

    效果如下：

  • 引用

  • 多层嵌套引用

  • 很多层( ..真的够了 )

敲重点：”>“后有空格！！！最好每个都打，但貌似打在最后也是可以的（ …嫌弃 ）

4. 修改字色、字号、字体

这是我最喜欢的环节！[ 谁能拒绝花里胡哨呢？ ]语法如下：

  • <font color=#0000ff>修改字色</font>

  • <font size=5>修改字号</font>

  • <font face=”楷体”>修改字体</font>

    效果如下：

  • 修改字色

  • 修改字号

  • 修改字体

敲重点：”字体”的引号为英文！！！如果同时修改字色字号字体的话，可以套用这个格式：

​ <font color=#0000ff size=5 face=”楷体”>同时修改</font>

​ 效果如下：

​ 同时修改

​ 对了字体颜色建议使用web安全色网页安全色大全 (shejidaren.com)
